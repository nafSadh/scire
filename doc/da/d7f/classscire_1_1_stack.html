Detailed Description

template<typename Type, typename SzType = int>
class scire::Stack< Type, SzType >

Stack class.

Linked list implementation of stack. A top pointer points to first item in the stack. Each item points to next item of it and the last item points to nullptr. Items can be added (push) and removed (pop) at top of the stack.

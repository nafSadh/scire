Detailed Description

template<typename Type, typename SzType = int>
class scire::SinglyList< Type, SzType >

A Singly Linked List class.

Singly linked list contains a linear list of items. Each item points to it's next; the last item of the list points to nullptr. A head pointer points to the first element of the list.
